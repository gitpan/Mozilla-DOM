=head1 NAME

Mozilla::DOM::NSHTMLAreaElement

=for object Mozilla::DOM::NSHTMLAreaElement

Mozilla::DOM::NSHTMLAreaElement is a wrapper around an instance of Mozilla's
nsIDOMNSHTMLAreaElement interface. This class inherits from
L<Supports|Mozilla::DOM::Supports>.
