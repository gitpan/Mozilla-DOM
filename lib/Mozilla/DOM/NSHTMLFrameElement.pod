=head1 NAME

Mozilla::DOM::NSHTMLFrameElement

=for object Mozilla::DOM::NSHTMLFrameElement

Mozilla::DOM::NSHTMLFrameElement is a wrapper around an instance of Mozilla's
nsIDOMNSHTMLFrameElement interface. This class inherits from
L<Supports|Mozilla::DOM::Supports>.
