=head1 NAME

Mozilla::DOM::Window2

=head1 DESCRIPTION

Mozilla::DOM::Window2 is a wrapper around an instance of Mozilla's
nsIDOMWindow2 interface. This class inherits from
L<Window|Mozilla::DOM::Window>.
