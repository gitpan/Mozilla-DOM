=head1 NAME

Mozilla::DOM::NSHTMLAnchorElement

=for object Mozilla::DOM::NSHTMLAnchorElement

Mozilla::DOM::NSHTMLAnchorElement is a wrapper around an instance of Mozilla's
nsIDOMNSHTMLAnchorElement interface. This class inherits from
L<Supports|Mozilla::DOM::Supports>.
