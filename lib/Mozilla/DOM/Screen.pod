=head1 NAME

Mozilla::DOM::Screen

=for object Mozilla::DOM::Screen

Mozilla::DOM::Screen is a wrapper around an instance of Mozilla's
nsIDOMScreen interface. This class inherits from
L<Supports|Mozilla::DOM::Supports>.
