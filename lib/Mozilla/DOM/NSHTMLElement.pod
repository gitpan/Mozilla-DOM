=head1 NAME

Mozilla::DOM::NSHTMLElement

Mozilla::DOM::NSHTMLElement is a wrapper around an instance of Mozilla's
nsIDOMNSHTMLElement interface. This class inherits from
L<Supports|Mozilla::DOM::Supports>.
