=head1 NAME

Mozilla::DOM::NSHTMLFormElement

=for object Mozilla::DOM::NSHTMLFormElement

Mozilla::DOM::NSHTMLFormElement is a wrapper around an instance of Mozilla's
nsIDOMNSHTMLFormElement interface. This class inherits from
L<Supports|Mozilla::DOM::Supports>.
