=head1 NAME

Mozilla::DOM::DocumentRange

=for object Mozilla::DOM::DocumentRange

Mozilla::DOM::DocumentRange is a wrapper around an instance of Mozilla's
nsIDOMDocumentRange interface. This class inherits from
L<Supports|Mozilla::DOM::Supports>.

 * The nsIDOMDocumentRange interface is an interface to a document
 * object that supports ranges in the Document Object Model.
 *
 * For more information on this interface please see
 * http://www.w3.org/TR/DOM-Level-2-Traversal-Range/
 *
 * @status FROZEN
