=head1 NAME

Mozilla::DOM::Location

=for object Mozilla::DOM::Location

Mozilla::DOM::Location is a wrapper around an instance of Mozilla's
nsIDOMLocation interface. This class inherits from
L<Supports|Mozilla::DOM::Supports>.
